# `utility/`
This folder contains all the utility and supportive files required for the repository.




### `mlb_team_colors.json`
This JSON file is a dictionary mapping the MLB team abbreviations to a list of 3 integers, representing the team's RGB color.


### `nba_team_colors.json`
This JSON file is a dictionary mapping the NBA team abbreviations to a list of 3 integers, representing the team's RGB color.


### `nfl_team_colors.json`
This JSON file is a dictionary mapping the NFL team abbreviations to a list of 3 integers, representing the team's RGB color.


### `nhl_team_colors.json`
This JSON file is a dictionary mapping the NHL team abbreviations to a list of 3 integers, representing the team's RGB color.


### `team_abbrs.json`
This JSON file is a dictionary mapping team names to a 2-4 character team abbreviation. Team names are the strings that OddsPortal uses, with spaces and periods replaced with underscores and all letters lowercased.
